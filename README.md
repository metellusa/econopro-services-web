# EconoPro Services Website

A modern marketing site for **EconoPro Services** built with **Vite + React + Tailwind CSS**.


## Tech Stack

- Vite
- React
- Tailwind CSS
- Lucide React icons

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Deploy to Netlify

1. Upload the project folder or connect the repo to Netlify.
2. Use the following build settings:
   - **Build command:** `npm run build`
   - **Publish directory:** `dist`
3. Netlify Forms is already wired into both forms on the site.
4. Replace the footer placeholders for phone number and email with your real contact info.
